# writevoid.com

[![Website Carbon](https://img.shields.io/badge/Website_Carbon-0.01g_CO2-brightgreen?style=flat-square)](https://www.websitecarbon.com/website/writevoid-com/)

writevoid.com is a small web project designed to help you focus on writing. Fade timers with two different fading timers. Faded and removed text is not saved, not possible to recover.
User can set word limit, after which download-button will appear that makes possible to download text in txt-format. Fade timer is stopped after download button is pressed and continue when typing continues.

## Privacy

writevoid does not use trackers, ads, or cookies. It has no backend, can run fully locally, and can also work well as a PWA.
